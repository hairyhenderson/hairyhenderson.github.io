Dec 13, 2015 · 1 minute read · Comments
electronicssites

It’d be cool to use the Wiimote and/or nunchuck as an input device to control I dunno, a drone or something?

Haven’t gotten very far, but here’s some links for posterity:

  • wiibrew - lots of in-depth info, like docs for the various protocols and data formats, esp. around the ‘mote and nunchuck
  • hackaday category - relevant articles
  • gobot lib - i2c for the nunchuck
  • nunchucky from Adafruit - breakout for nunchuck and other devices (like the classic controller) to use them directly
